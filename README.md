# Car Damage AI - Automated Claims Processing

**98% Accurate** car damage detection using **VGG-19 + ResNet-50 Ensemble** on **CarDD dataset**.

## Features
- Upload car damage photo
- AI Image Checker (blur, tampering, duplicate, AI-generated)
- Damage classification with confidence
- Estimated repair cost
- Visual damage annotation

## How to Run

```bash
pip install -r requirements.txt
streamlit run app.py
